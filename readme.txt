Add Icons and loading screens for various versions of iPhone and iPad.

== Description ==

Simply add the Icons and loading screens for various versions of iPhone and iPad.

Features:-
1. Easy to manage.
2. It offers and demonstrated the user experience for icons with and without the Apple icon glossy effect.
3. Supports to all apple devices.
4. You can set small(57x57) and large(114x114) icons for various devices.
5. You can set loading screens also for all apple devices.
For further help contact us at [http://devx.in](http://devx.in) [http://developerextensions.com](http://developerextensions.com) [http://sunilkumardhanda.me](http://sunilkumardhanda.me)

== Installation ==

1. Upload the Appleicons plugin to your plugin directory wp-content/plugins/. 
2. Activate it via the 'Plugins' menu in WordPress.
3. Upload the Icons and loading screens which you want to show on iphone during loading screen.

You can also installed directly from the main WordPress Plugin page.

1. Go to the Plugins => Add New page.
2. Enter 'appleicons' in the textbox and click the 'Search Plugins' button.
3. In the list of relevant Plugins click the 'Install' link for Simple Sitemap on the right hand side of the page.
4. Click the 'Install Now' button on the popup page.
5. Click 'Activate Plugin' to finish installation.
6. Goto settins/Apple Icons menu in WordPress.
7. Upload the Icons and loading screens which you want to show on iphone during loading screen.
